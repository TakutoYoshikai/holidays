### Requirements
* macOS or Ubuntu
* Node.js
* npm

### Usage
**install**
```bash
npm install -g TakutoYoshikai/holidays
```

**use command**
```bash
holidays
# It shows nothing if it's not a holiday.
```
You can show figlet message when you open terminal by adding `holidays` to .bashrc
